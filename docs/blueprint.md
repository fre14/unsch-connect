# **App Name**: UNSCH Connect

## Core Features:

- Registration and Authentication: Allow students to register using their @unsch.edu.pe email address and student code.
- Content Publishing: Enable users to publish photos, videos, text, and polls to various boards.
- Institutional Announcements: Provide a dedicated space for official announcements from the university, schools, and faculty.
- Community Interaction: Allow students to interact with each other's content through comments, reactions, shares, and reports.
- Academic Schedule Management: Enable students to create and manage their academic schedules, including personal events and reminders.
- Information Search: Implement a search bar to find users, posts, courses, activities, and announcements.
- Account Management: Allow users to manage their profile information, privacy settings, and notifications.

## Style Guidelines:

- The concept of connecting students guides color choice. We will choose analogous colors, aiming for serenity and security in a dark scheme, so that the focus stays on shared information and community. Primary color: Medium-dark cyan (#468189).
- Background color: Very dark cyan, almost black (#1A2324).
- Accent color: Bright teal (#2EC4B6) for interactive elements and highlights.
- Body: 'Inter', a sans-serif font known for its modern and neutral appearance.
- Headings: 'Space Grotesk', a geometric sans-serif with a techy feel. 'Space Grotesk' will also be used for all other text when a shorter chunk of text is expected
- Use clean, modern icons to represent different actions and sections within the app.
- Employ a tab-based navigation system for easy access to different sections of the app.
- Incorporate subtle transitions and animations to enhance user experience, such as loading screens and interactive feedback.
